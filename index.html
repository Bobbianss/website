Pyramiden 2801

A project made for the Game Design course of Politecnico di Torino (2020/2021).

History

Pyramiden was a small town in the Svalbard islands founded by Swedish miners in 1910.

But at the end of the 90s an airplane crashed into the city, bringing Pyramiden to their knees. By 1998 every residents had left the city.

Pyramiden became a ghost town.

But somebody didn’t give up.

Casper, the old wiseman, has left his legacy in Pyramiden: he hid in the arcade room the key to rianimate the city.

It’s time to bring Pyramiden back to life.

 

Languages and Technology used:

  • C#
  • Unity Environment
  • Blender
  • Adobe Audition / Illustrator / Premiere
  • Substance

Sketchizer

A project made for the Computer Vision course of Politecnico di Torino (2020/2021).

Sketchizer is a program that enables you to turn an image into a hand drawing-like image in .svg format, using python and openCV.

The app’s purpose is to quickly generate a prototype that could be further manually edited using a .svg editor tool; or just to directly transform an image into a simpler outline.

 

 

Languages and Technology used:

  • Python
  • OpenCv Library

High School – Final Project

My project is an overview that covers the robotic arm subject. On one hand, it focuses on the second level robots. In particularly, it combines the features of the first level robot with the second level robot. On the other hand, it deals with the main steps regarding robots’ innovation.

The project it had been built from scratch. The materials like wood and iron had been recycled, while the particular sensors were bought by me.

 

 

Languages and Technology used:

  • Arduino
  • PHP, CSS, HTML
  • Javascript
